# WEB-016 — watch 页改 3 列固定布局

**优先级**：P1（视频体验 + 相关视频信息架构）
**区域**：web / layout
**依赖**：WEB-015 ✅（app-shell 宽度令牌）、WEB-014 ✅（BackLink）

---

## 背景

PM 实测反馈两个问题：

1. **视频随窗口宽度变大变小**——已被 Codex1 用 `lg:max-w-[48rem]` (768px) 固定，但**左列右侧出现空荡**（列还是 `lg:basis-[63%]` 占着但视频不再填满）
2. **相关视频面板是悬浮覆盖层**——hover/pin 触发从右边滑出，会盖住字幕区。PM 不希望相关视频"弹在字幕里"，要求独立成第三列贴右

## 目标

改成清晰的 3 列固定布局：视频+章节（左）/ 字幕（中）/ 相关视频（右）。

---

## 设计（Claude2 二审通过版 2026-05-22）

### 列宽分配（1920px 视口 / max-w-app-shell=1536 / pl-7=28）

```
┌──────────────────┬─────────────────┬─────────────┐
│ 视频 + 章节        │ 字幕             │ 相关视频     │
│ 768px (48rem)    │ 480px (flex-1)  │ 260px       │
│ 视频左对齐填满     │ 西+中 双列      │ 常驻无折叠   │
└──────────────────┴─────────────────┴─────────────┘
                          1536px 总宽
```

**数学**：1536 − 28(pl-7) − 768(左) − 260(右) = **480px**（中列字幕）。这个数算错过——PM 一开始估 568px、Claude2 算出实际 423px、用右列从 290 压到 260 救回 480px。

### 左列（视频 + meta + 章节）

```tsx
<section className="flex flex-col px-4 py-4 lg:basis-[48rem] lg:shrink-0 lg:justify-start lg:overflow-y-auto lg:px-0 lg:py-8 lg:pr-6">
```

- **`lg:basis-[51rem]` → `lg:basis-[48rem]`**（768px，紧贴视频不留空）
- 加 **`lg:shrink-0`** 确保不被压缩
- 保留 `lg:justify-start lg:overflow-y-auto`（之前修过的）

视频容器：

```tsx
<div className="w-full overflow-hidden rounded-surface bg-black shadow-elevated lg:mt-2 lg:max-w-[48rem]">
```

- **删掉 `lg:mx-auto`**（之前加错了）—— 列基础宽 = 视频宽，视频自然填满，居中没意义
- 保留 `lg:max-w-[48rem]`（视频不超过 768px）

### 中列（字幕）

保持现状不变：
```tsx
<section className="h-[60vh] min-w-0 border-t border-gray-200 bg-surface lg:h-auto lg:flex-1 lg:border-l lg:border-t-0">
  <TranscriptPanel ... />
</section>
```

`lg:flex-1` 自动占用剩余空间（480px）。

### 右列（相关视频 — 改造重点）

```tsx
<aside className="hidden border-l border-gray-200 bg-surface lg:flex lg:w-[260px] lg:shrink-0">
  <RelatedPanel relatedVideos={relatedVideos} />
</aside>
```

**改动**：
- `<div className="hidden lg:block">` → `<aside className="hidden border-l border-gray-200 bg-surface lg:flex lg:w-[260px] lg:shrink-0">`
- 固定 260px 宽（不是 290px——为了救字幕宽）
- `lg:shrink-0` 防被压缩
- 移动端不变（仍 `hidden` 不显示）

### RelatedPanel 组件改造（删 hover/pin 状态机）

`src/app/watch/RelatedPanel.tsx`：

**删掉**：
- `visible / pinned` state 和 setter
- `scheduleOpen / scheduleClose / clearTimers` 函数
- `showTimerRef / hideTimerRef` refs
- 边缘触发条 `<div className="absolute bottom-0 right-0 top-0 z-20 w-2.5">`
- 滑动 class `translate-x-full / translate-x-0`
- 钉住按钮 `<button>◀</button>` 及其逻辑

**保留**：
- 列表渲染（视频卡片）
- "相关视频" 头部
- "暂时还没有拉到相关视频" 空状态

**简化为**：
```tsx
"use client";  // 仍是 client 因为有 Link 但可以考虑改 server component

import Link from "next/link";
import type { YouTubeVideoPayload } from "@/lib/youtube-shared";
import { formatVideoDurationBadge } from "@/lib/youtube-shared";

type RelatedPanelProps = {
  relatedVideos: YouTubeVideoPayload[];
};

export function RelatedPanel({ relatedVideos }: RelatedPanelProps) {
  return (
    <div className="flex w-full flex-col">
      <div className="flex items-center border-b border-gray-100 px-3 py-3">
        <span className="text-[13px] font-semibold text-gray-700">相关视频</span>
      </div>
      <div className="flex-1 overflow-y-auto px-2 py-2">
        {relatedVideos.length === 0 ? (
          <div className="px-3 py-6 text-sm text-gray-400">暂时还没有拉到相关视频</div>
        ) : (
          relatedVideos.map((video) => (
            <Link
              className="flex gap-2 rounded-card px-2 py-1.5 transition hover:bg-gray-50"
              href={`/watch?v=${video.id}`}
              key={video.id}
            >
              <div className="relative h-[54px] w-[96px] shrink-0 overflow-hidden rounded-md bg-gray-200">
                <img alt="" className="h-full w-full object-cover" src={video.thumbnailUrl} />
                <span className="absolute bottom-1 right-1 rounded bg-black/80 px-1.5 py-0.5 text-[10px] font-semibold text-white">
                  {formatVideoDurationBadge(video.durationSec)}
                </span>
              </div>
              <div className="min-w-0 pt-0.5">
                <p className="line-clamp-2 text-[12.5px] font-medium leading-5 text-gray-900">
                  {video.title}
                </p>
                <p className="mt-1 text-[11px] text-gray-400">{video.channelTitle}</p>
              </div>
            </Link>
          ))
        )}
      </div>
    </div>
  );
}
```

**Claude2 P2 优化（必做）**：
- 缩略图从 108×60 → **96×54**
- 列表 padding 从 `px-2 py-3` → `px-2 py-2`
- 卡片 padding 从 `px-2 py-2` → `px-2 py-1.5`

密度提升 ~15%，260px 列宽容下足够多条相关视频。
