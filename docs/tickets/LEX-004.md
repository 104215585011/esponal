# LEX-004 — 回补 seed 阶段 skipped 的短语条目（backlog）

**类型**：后端 / 数据补全
**实现**：Codex1（待启动）
**前置依赖**：无（独立）
**预估**：0.5 天
**优先级**：低（backlog，301 条短语已在线够用；缺的多为 B1+ 习语）

---

## 背景

LEX-001 Phase 3 批量生成短语语料时，seed CSV（`data/phrases-a1-a2-seed.csv`）计划入库 457 条（319 多 token + 138 单 token）。实际 DB 现有 301 条 phrase-kind。多 token 里有 **23 条没进库**——经核查，这些**系统性地是 Tatoeba 平行语料没有例句的条目**（尤其习语），seed 脚本按"每条错误隔离 + 跳过"策略略过了它们。没有保留 skip 日志，本清单由 PM 2026-05-29 对比 seed CSV ↔ DB 重建。

> 注：单 token 那 ~138 条不是 skip，是被 LEX-CLEANUP-001 故意迁/删的（误分类），与本 ticket 无关。

---

## 缺失清单（23 条）

### A. 实用搭配（7，建议补）
- `acostumbrarse a` 习惯于
- `quejarse de` 抱怨
- `arrepentirse de` 后悔
- `tener vergüenza` 害羞
- `tener la culpa` 有责任/怪…
- `poner en marcha` 启动/发动
- `ir + gerundio` 逐渐…（带 `+`，是句型非固定词组，tokenize/匹配需特殊处理或排除）

### B. 课堂问句（3，建议补）
- `¿puede repetir?` 能再说一遍吗
- `¿cuál es la tarea?` 作业是什么
- `¿en qué página estamos?` 我们在第几页

### C. 习语（11，偏 B1+，A1-A2 试点价值低）
- `estar de mal humor` 心情不好
- `no dar pie con bola` 什么都做不对
- `quedarse de piedra` 惊呆
- `andar con rodeos` 拐弯抹角
- `no pegar ojo` 一夜没合眼
- `ser un cero a la izquierda` 无足轻重
- `tener mano izquierda` 善于处理棘手事
- `irse al traste` 泡汤/失败
- `no dar crédito` 不敢相信
- `tener un humor de perros` 脾气很坏
- `ser un trozo de pan` 心地善良

> 另：检查时 `muy bien, gracias` / `más despacio, por favor` 两条疑似因 CSV 含逗号字段解析而误报缺失，启动本 ticket 时需先确认它们到底在不在库。

## 实施建议（启动时）

1. 不要重跑完整 Tatoeba 管线——缺的就是 Tatoeba 没例句的，会再次 skip
2. 写一次性脚本 `scripts/lexicon/backfill-skipped-phrases.mjs --dry-run/--write`
3. 直接用 DeepSeek 给目标条目**生成例句 + 用法说明**（跳过 Tatoeba），upsert 入库
4. `ir + gerundio` 这条评估：要么按句型特殊存、要么从清单剔除
5. 入库后复核 DB phrase-kind 计数 + 抽查例句质量
