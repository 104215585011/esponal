# Esponal 移动端设计语言(干净现代·极简)— 全站统一基准

**确立**：2026-06-03,用户批准 MOBILE-003 v3 模型(`docs/tickets/MOBILE-003-mockup.html`)后提炼。
**用途**：所有移动端页面(MOBILE-003~008 及后续)的视觉**统一基准**。每页设计/实现都必须遵守本语言,避免各做各、跑偏成"丑"。**看 v3 模型 = 看标准。**

## 0. 总基调
**干净、现代、极简、轻盈**(对标 DejaVocab)。浅色大留白、清爽无衬线、克制配色、轻卡片。**翡翠绿只做点缀,不铺满。** 反掉:暖纸/宋体/复古、纯白描边盒子、通用 SaaS 味、平。

## 1. 颜色 token
| 名 | 值 | 用途 |
|---|---|---|
| 底 | `#ffffff`(纯白)/ soft `#f7f8f7` | 页底 |
| 墨 | `#121413`(ink)/ `#3f433f`(ink-2) | 标题/正文 |
| 灰 | `#8a8f88`(muted) | 副文/说明 |
| 线 | `#eceeec` | 细分隔/卡边 |
| 品牌 | `#10b981`(brand)/ `#059669`(deep) | 强调,点缀 |
| 品牌软 | `#ecf8f2`(brand-soft) | 徽标/进度 chip 底 |
| 阴影 | `0 1px 2px rgba(16,24,20,.04), 0 8px 24px -16px rgba(16,24,20,.18)` | 卡片软阴影 |
> **禁 sky 蓝 / purple。** 强调色只有翡翠绿。映射到项目 token:brand=brand-500/600,灰=zinc 系。

## 2. 字体
- **拉丁 / 数字**:`Plus Jakarta Sans`(清爽现代,序号/数字/数据用它)。
- **中文**:`Noto Sans SC`(300 轻 / 400 / 500 / 700)。
- **不用衬线、不用 Inter/系统字**。标题 700、正文 300-400。
> 项目落地:若已有等价字体栈则复用;数字/序号务必用几何无衬线(Plus Jakarta Sans 类)。

## 3. 间距 / 圆角 / 节奏
- 页左右 padding:**22px**。区块间距 `margin-top:38px`。
- 卡片圆角 **20px**;按钮/统计块 **14-16px**;头像 50%。
- 标题 18-33px(页主标 33,区标 18)、正文 13-14px、说明 11.5-13px。
- 大留白,别挤。

## 4. 组件范式(照抄)
- **顶栏**:`sticky` 毛玻璃 `rgba(255,255,255,.78)+blur(16px)` + 底细线;左头像(翡翠圆 36px)/ 中订阅 / 右搜索图标。
- **Hero**:**纯白、不堆渐变面板**。小问候(翡翠点 + 文字)→ 大标题(700,关键词翡翠)→ 灰说明 → **翡翠实心 CTA**(圆角 14、`gap` + 箭头、软翡翠阴影)。
- **统计**:分格卡(白底细边),大数字用 Plus Jakarta 700 + 小单位灰字 + 下方 label。
- **横滑卡(学习路径类)**:`scroll-snap` 横滑;卡 = 白底 + 细边 + 软阴影 + 圆角 20;**翡翠数字小徽标**(圆角方块 38px,brand-soft 底 + deep 字);标题 600;说明 muted 300;进度 chip(brand-soft);底部细分隔线 + "进入学习" + 翡翠箭头。
- **区标题**:18px 700 ink + 右侧 muted "全部/更多"。
- **底部 tab**:毛玻璃,当前态翡翠 deep + 700,其余 muted。
- **载入动画**:分层 `translateY+opacity` 错峰淡入(delay .04/.12/.2)。

## 5. 原则
- 强调色克制(一屏别超过几处翡翠)。
- 卡片要么细边、要么软阴影,**别又粗边又重阴影**。
- 触摸目标≥44px、安全区 `.pb-safe`、内容给固定顶栏+底部 tab 留 padding。
- 桌面端不回退(只改移动 <768px,`md:` 还原)。

> **每页设计/实现前,先看 `MOBILE-003-mockup.html` 对齐手感,再照本语言展开。**
